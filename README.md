# Roguelike-game
University computer graphics project. The object of the game is to find a way out of a maze with traps and enemies.
## Launching
```
cmake CMakeLists.txt
make
./bin/main
```
## Gameplay
- WASD - movement
- E - attack
- Black blocks - doors to other rooms
- Blue block - exit
- Fire blocks - death
- Enemies and traps - takes 1 health
